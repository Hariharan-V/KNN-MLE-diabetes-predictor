# KNN-MLE-diabetes-predictor

#Dependencies
Python 2.7, Numpy

#Instructions
run "python knn.py" to get an overall summary of accuracy given K = 1,5,11 neighbors
run "python mle.py" to get an overall summary of accuracy using the MLE method
